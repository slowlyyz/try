# Cesium for Satellite Orbit


## Frontend

- Node

+ Version: 20.18.0
+ Multiple version management tool: `nvm`

- Vue

+ Build tool: `vite`
+ Language: `TypeScript`
+ HTTP request: `axios`

- How to run the project

`npm install`
`npm run dev`

- Reference

+ Cesium Documentation

[Polyline](https://cesium.com/learn/cesiumjs/ref-doc/Polyline.html)

+ Cesium Sandcastle

[Polyline](https://sandcastle.cesium.com/?src=Polyline.html&label=All)

+ 3D Models

[Sketchfab](https://sketchfab.com/)


## Backend

- Python

+ Package management tool: `pdm`
